# Epic 01 — 인증 & 온보딩

**포함 기능:** F1 (회원가입/로그인), F13 (개인정보 동의 — 진입 게이트), F14 (7일 트라이얼)  
**선행 조건:** 없음 (첫 번째 에픽)  
**완료 기준:** 가입 → 트라이얼 자동 시작 → 홈 진입 전체 플로우 동작

---

## Story 1 — 개인정보 동의 화면

**As a** 앱 첫 실행 유저  
**I want** 목소리 수집에 대한 동의를 명확히 확인하고 싶다  
**So that** 내 생체정보가 어떻게 처리되는지 알고 가입할 수 있다

### 태스크 체크리스트

- [ ] 개인정보 동의 화면 UI (수집 항목, 보관 기간, 즉시 삭제 방침 명시)
- [ ] 필수 동의 항목 체크박스 + 전체 동의 토글
- [ ] 개인정보처리방침 전문 링크 (웹뷰 또는 외부 URL)
- [ ] 미동의 시 가입 화면 진입 차단 + 종료 안내 다이얼로그
- [ ] 동의 완료 시 로컬 플래그 저장 (재실행 시 재노출 방지)

### 수용 기준

- Given 앱 최초 실행 / When 동의 화면 진입 / Then 필수 항목 미체크 상태로 노출, 가입 버튼 비활성
- Given 필수 항목 모두 체크 / When 다음 버튼 탭 / Then 회원가입 화면 이동
- Given 동의 거부(취소) / When 확인 / Then "앱 사용을 위해 동의가 필요해요" 안내 + 앱 종료 옵션
- Given 앱 재실행 / When 이미 동의 완료 / Then 동의 화면 스킵 → 로그인/홈 직행

---

## Story 2 — 회원가입 (이메일)

**As a** 신규 유저  
**I want** 이메일로 계정을 만들고 싶다  
**So that** 앱을 개인 계정으로 사용할 수 있다

### 태스크 체크리스트

- [ ] 이메일 + 비밀번호 입력 폼 UI
- [ ] 이메일 유효성 검증 (형식)
- [ ] 비밀번호 강도 규칙 (8자 이상, 숫자+문자 조합)
- [ ] 이메일 중복 체크 → "이미 등록된 이메일" 에러 + 로그인 버튼 노출
- [ ] 가입 완료 → 이메일 인증 발송 (선택: V1에서 인증 강제 여부 결정)
- [ ] 가입 성공 → F14 트라이얼 자동 시작 → 홈 이동

### 수용 기준

- Given 이메일 형식 오류 / When 다음 탭 / Then 인라인 에러 메시지
- Given 이미 등록된 이메일 / When 가입 시도 / Then 에러 + 로그인 유도 버튼
- Given 정상 가입 완료 / When 완료 / Then 트라이얼 배지와 함께 홈 화면 이동

---

## Story 3 — 회원가입 (소셜: Apple / Google)

**As a** 신규 유저  
**I want** 소셜 계정으로 빠르게 가입하고 싶다  
**So that** 이메일/비밀번호 없이 간편하게 시작할 수 있다

### 태스크 체크리스트

- [ ] Apple Sign-in 연동 (iOS 필수)
- [ ] Google Sign-in 연동 (iOS/Android)
- [ ] 소셜 토큰 → 서버 인증 → JWT 발급
- [ ] 기존 소셜 계정 재로그인 시 계정 연결 (신규 생성 안 함)
- [ ] 소셜 로그인 실패(토큰 만료/취소) → 재시도 안내

### 수용 기준

- Given Apple 로그인 탭 / When 인증 완료 / Then 7일 트라이얼 + 홈 이동
- Given 기존 소셜 계정 재로그인 / When 완료 / Then 기존 음원 목록 복원
- Given 소셜 로그인 실패 / When 오류 / Then 에러 메시지 + 재시도 버튼

---

## Story 4 — 로그인

**As a** 기존 유저  
**I want** 이메일 또는 소셜로 로그인하고 싶다  
**So that** 내 계정과 음원에 다시 접근할 수 있다

### 태스크 체크리스트

- [ ] 이메일 로그인 폼 UI
- [ ] 비밀번호 찾기 (이메일 재설정 링크 발송)
- [ ] 소셜 로그인 버튼 (가입 화면과 동일 컴포넌트)
- [ ] 세션 유지 (자동 로그인, 토큰 갱신)
- [ ] 세션 만료 시 로그인 화면 리다이렉트 (음원 데이터 유지)

### 수용 기준

- Given 올바른 자격증명 / When 로그인 / Then 홈 화면 이동 + 기존 음원 목록 복원
- Given 잘못된 비밀번호 / When 로그인 시도 / Then "이메일 또는 비밀번호를 확인해주세요" 에러
- Given 앱 재실행 (세션 유효) / When 실행 / Then 로그인 없이 홈 직행

## Story 5 — 7일 무료 Premium 트라이얼

**As a** 신규 가입 유저  
**I want** 가입 즉시 7일간 Premium 기능을 무료로 써보고 싶다  
**So that** 구독 결정 전에 앱의 가치를 경험할 수 있다

### 태스크 체크리스트

- [ ] RevenueCat 트라이얼 상품 설정 (7일 무료 → 유료 자동 전환)
- [ ] 가입 완료 시 트라이얼 자동 시작 (별도 버튼 없음)
- [ ] 홈/재생 화면 트라이얼 배지 ("7일 무료 체험 중 · N일 남음")
- [ ] 트라이얼 D-1 푸시 알림 ("내일 무료 체험이 끝나요")
- [ ] 트라이얼 만료 → 무료 플랜 자동 다운그레이드 + 구독 유도 화면

### 수용 기준

- Given 신규 가입 완료 / When 홈 진입 / Then 트라이얼 배지 + 광고 없음 + 백그라운드 허용
- Given 트라이얼 D-1 / When 알림 허용 / Then 푸시 알림 발송
- Given 트라이얼 만료 / When 결제 없음 / Then 배너 광고 노출 + 구독 유도 팝업
- Given 트라이얼 만료 후 구독 결제 / When 완료 / Then 즉시 Premium 복원
