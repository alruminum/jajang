# Epic 06 — 개인정보 & 데이터 관리

**포함 기능:** F13 (개인정보 동의 + 목소리 샘플/결과물 삭제)  
**선행 조건:** Epic 01 완료 (회원가입)  
**완료 기준:** 유저가 설정에서 데이터 삭제 및 계정 탈퇴를 완료할 수 있는 플로우 동작

> **참고**: 개인정보 동의 화면(첫 실행 게이트)은 Epic 01 Story 1에 포함. 이 에픽은 설정 내 데이터 관리 기능 중심.

---

## Story 1 — 목소리 샘플 삭제 요청

**As a** 유저  
**I want** 서버에 저장된 내 목소리 샘플을 언제든 삭제할 수 있다  
**So that** 생체정보를 내 의지로 통제할 수 있다

### 태스크 체크리스트

- [ ] 설정 화면 > "내 데이터 관리" 섹션
- [ ] "목소리 샘플 삭제" 버튼 (서버에 샘플이 남아 있는 경우만 활성)
- [ ] 삭제 전 확인 다이얼로그 ("삭제하면 복구할 수 없어요")
- [ ] 삭제 요청 → 서버 처리 → 48시간 이내 완료 알림 (이메일 또는 인앱)
- [ ] 이미 생성 완료(샘플 없음) 상태라면 "이미 삭제되었어요" 안내

### 수용 기준

- Given 서버에 샘플 존재 / When 삭제 요청 / Then 48시간 이내 삭제 + 완료 알림
- Given 이미 삭제된 샘플 / When 버튼 탭 / Then "이미 삭제되었어요" 안내
- Given 삭제 완료 / When 설정 재진입 / Then 버튼 비활성 상태

## Story 2 — 생성 음원 삭제

**As a** 유저  
**I want** 내가 만든 자장가 음원을 삭제할 수 있다  
**So that** 불필요한 파일을 정리하거나 개인 데이터를 삭제할 수 있다

### 태스크 체크리스트

- [ ] 홈 음원 카드: 스와이프 또는 롱탭 → 삭제 옵션
- [ ] 삭제 확인 다이얼로그 ("삭제하면 복구할 수 없어요")
- [ ] 클라이언트 로컬 캐시 삭제 + 서버 파일 삭제 + DB 레코드 삭제
- [ ] 삭제 후 홈 목록에서 즉시 제거
- [ ] 설정 > "내 데이터 관리" > "모든 음원 삭제" 일괄 삭제 옵션

### 수용 기준

- Given 음원 삭제 요청 / When 확인 / Then 로컬 + 서버 동시 삭제 + 목록에서 제거
- Given 일괄 삭제 / When 확인 / Then 전체 음원 삭제 + 홈 빈 상태 UI
- Given 오프라인 상태 / When 삭제 요청 / Then 로컬만 즉시 삭제, 서버는 온라인 복귀 후 처리

## Story 3 — 계정 탈퇴 & 전체 데이터 삭제

**As a** 유저  
**I want** 계정을 탈퇴하고 모든 내 데이터를 삭제할 수 있다  
**So that** 더 이상 서비스를 쓰지 않을 때 흔적을 남기지 않을 수 있다

### 태스크 체크리스트

- [ ] 설정 > "계정" > "계정 탈퇴" 버튼
- [ ] 탈퇴 전 2단계 확인 (사유 선택 선택사항 + 최종 확인)
- [ ] 탈퇴 처리: 목소리 샘플 + 생성 음원 + 계정 정보 모두 삭제
- [ ] 클라이언트 로컬 데이터(캐시, 설정) 즉시 삭제
- [ ] 탈퇴 완료 → 로그인 화면 이동
- [ ] 구독 중인 경우 탈퇴 전 구독 취소 안내 (앱스토어/플레이스토어에서 별도 처리 필요)

### 수용 기준

- Given 탈퇴 완료 / When / Then 모든 서버 데이터 삭제 + 로그인 화면 이동
- Given 탈퇴 완료 / When 로컬 캐시 / Then 즉시 삭제
- Given 구독 중 탈퇴 시도 / When / Then 구독 취소 먼저 하라는 안내 노출

## Story 4 — 개인정보처리방침 & TOS 접근

**As a** 유저  
**I want** 언제든 개인정보처리방침과 이용약관을 확인하고 싶다  
**So that** 내 데이터가 어떻게 처리되는지 투명하게 알 수 있다

### 태스크 체크리스트

- [ ] 설정 > "법적 정보" 섹션 (개인정보처리방침 / 이용약관)
- [ ] 외부 URL 또는 인앱 웹뷰로 표시
- [ ] TOS 내 명시 항목:
  - 제3자 목소리 업로드 금지
  - 유튜브 등 외부 업로드 금지
  - 음성 데이터 수집·삭제 방침
  - 의료기기 아님 고지

### 수용 기준

- Given 설정 법적 정보 탭 / When 개인정보처리방침 탭 / Then 해당 문서 노출
- Given TOS 문서 / When 열람 / Then 제3자 목소리 금지 조항 포함 확인
